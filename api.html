— 配置参数

参数名 类型 说明
currPage int 配置当前页，默认为1.
pageCount int 配置总页数，默认为10.
pageNumber int 配置页码数量.默认为5.
cssStyle string 配置样式属性.
ajax object 配置ajax参数.
panel object 配置显示面板参数.

ajax(object) 类型 说明
on boolean 配置ajax是否开启,默认为false.
url string 配置请求路径.
type string 配置请求类型,提供二种选项."POST","GET".
dataType string 配置请求数据类型,提供三种选项."json","xml","html".
pageCountId string 配置后台返回总页数的参数名.
param string 配置ajax请求参数,值以key-value形式.
ajaxStart function 配置ajax请求前调用该方法.
ajaxStop function 配置ajax结束后调用该方法.
onClick function(page) 配置点击某一页时，触发该事件.
callback function(data) 配置回调函数,data为回调的数据.

panel(object) 类型 说明
first string 配置首页显示值.
last string 配置尾页显示值.
next string 配置下一页显示值.
prev string 配置上一页显示值.
first_on boolean 配置首页是否显示.默认true,显示.
last_on boolean 配置尾页是否显示.默认true,显示.
next_on boolean 配置下一页是否显示.默认true,显示.
prev_on boolean 配置上一页是否显示.默认true,显示.
links string 配置鼠标浮动插件上，浏览器提示.
tipInfo_on boolean 配置提示信息.默认false,不显示.
tipInfo string 配置提示信息，提供两种类型{input}{select}.
tipInfo_css object 配置面板中输入框样式.值以key-value形式.

 

— 扩展方法

方法名称 参数 返回 说明
getPage() null 返回当前组件的当前页数. 得到当前页数.
onReload() null null 无条件重新刷新，适用于删除某条数据.
jumpPage(page) string null 跳转到某页.
onLoad(params) string null 加载插件,按照指定参数.
params值是key-value的键值对:
keywords=&workyear=99

— 组合插件

目前myPagination的核心组合插件：msgbox(消息提示),jTemplates(模板引擎).
可以扩展其他的插件：jqtransform(表单美化),fancybox(弹出层插件),都可以兼容.
